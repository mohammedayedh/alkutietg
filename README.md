- ✅ دعم كامل للغتين العربية والإنجليزية مع RTL/LTR

- ✅ نماذج تواصل واستفسارات
- ✅ عرض الفروع والخدمات والشراكات

## 🛠️ التقنيات المستخدمة

### Backend:
- **Django 6.0.2** - إطار عمل Python القوي
- **SQLite** - قاعدة البيانات (قابلة للترقية لـ PostgreSQL/MySQL)
- **Pillow** - معالجة الصور

### Frontend:
- **Bootstrap 5.3** (RTL/LTR) - تصميم متجاوب
- **Font Awesome 6.4** - الأيقونات
- **AOS Library** - الحركات والتأثيرات
- **Google Fonts (Cairo)** - خط عربي احترافي

### Production:
- **Gunicorn** - WSGI HTTP Server
- **WhiteNoise** - خدمة الملفات الثابتة
- **PostgreSQL** - قاعدة البيانات للإنتاج

## 🚀 التثبيت والتشغيل المحلي

### 1. استنساخ المشروع:
```bash
git clone https://github.com/mohammedayedh/alkutietg.git
cd alkutietg
```

### 2. تفعيل البيئة الافتراضية:
```bash
# Windows
venv\Scripts\activate

# Linux/Mac
source venv/bin/activate
```

### 3. تثبيت المتطلبات:
```bash
pip install -r requirements.txt
```

### 4. تطبيق الـ Migrations:
```bash
python manage.py migrate
```

### 5. إدخال البيانات الأولية:
```bash
python setup_initial_data.py
```

### 6. إنشاء حساب المدير:
```bash
python manage.py createsuperuser
```

### 7. تشغيل السيرفر:
```bash
python manage.py runserver
```

### 8. زيارة الموقع:
- **الموقع الرئيسي:** http://127.0.0.1:8000/
- **لوحة الإدارة:** http://127.0.0.1:8000/admin/
