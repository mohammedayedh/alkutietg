# دليل البدء السريع 🚀

## خطوات تشغيل الموقع

### 1️⃣ تفعيل البيئة الافتراضية
```bash
venv\Scripts\activate
```

### 2️⃣ التأكد من تثبيت المتطلبات
```bash
pip install -r requirements.txt
```

### 3️⃣ إنشاء حساب المدير
```bash
python manage.py createsuperuser
```
سيطلب منك:
- اسم المستخدم (Username)
- البريد الإلكتروني (Email)
- كلمة المرور (Password)

### 4️⃣ تشغيل السيرفر
```bash
python manage.py runserver
```

### 5️⃣ فتح الموقع
افتح المتصفح وانتقل إلى:
- **الموقع:** http://127.0.0.1:8000/
- **لوحة الإدارة:** http://127.0.0.1:8000/admin/

## 🎨 تخصيص المحتوى

### من لوحة الإدارة يمكنك:

#### 1. تعديل معلومات الشركة
- الذهاب إلى: **Company** → **معلومات الشركة**
- تعديل: الاسم، الشعار، الرؤية، الرسالة
- رفع: الشعار، فيديو الصفحة الرئيسية

#### 2. إدارة الفروع
- الذهاب إلى: **Company** → **الفروع**
- إضافة/تعديل/حذف الفروع
- تحديد نوع الفرع (رئيسي/إقليمي/شريك)

#### 3. إدارة الخدمات
- الذهاب إلى: **Services** → **الخدمات**
- إضافة خدمات جديدة
- رفع صور للخدمات

#### 4. إدارة الشراكات
- الذهاب إلى: **Services** → **الشراكات**
- إضافة شركاء استراتيجيين
- رفع شعارات الشركاء

#### 5. متابعة الرسائل
- الذهاب إلى: **Contact** → **رسائل التواصل**
- قراءة رسائل العملاء
- تغيير حالة الرسالة (جديد/قيد المعالجة/تم الحل)

## 🔧 أوامر مفيدة

### إنشاء نسخة احتياطية من قاعدة البيانات:
```bash
copy db.sqlite3 db.sqlite3.backup
```

### إعادة إدخال البيانات الأولية:
```bash
python setup_initial_data.py
```

### جمع الملفات الثابتة (للإنتاج):
```bash
python manage.py collectstatic
```

## 🌐 نشر الموقع (Production)

### قبل النشر:
1. تغيير `DEBUG = False` في `config/settings.py`
2. تعيين `SECRET_KEY` جديد وآمن
3. إضافة النطاق إلى `ALLOWED_HOSTS`
4. استخدام قاعدة بيانات أقوى (PostgreSQL/MySQL)
5. إعداد خادم ويب (Nginx/Apache)
6. استخدام Gunicorn أو uWSGI
