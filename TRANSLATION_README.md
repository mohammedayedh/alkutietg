# نظام الترجمة - دليل سريع 🌐

## ✅ تم التفعيل بنجاح!

تم تفعيل نظام الترجمة المتعدد اللغات في الموقع بنجاح.

---

## 🎯 كيفية الاستخدام

### 1. تشغيل الموقع
```bash
python manage.py runserver
```

### 2. تبديل اللغة
- افتح الموقع في المتصفح
- ابحث عن قائمة اللغة في أعلى الصفحة (Navbar)
- اختر اللغة: **العربية** أو **English**

---

## 📝 إضافة ترجمات جديدة

### الخطوة 1: إضافة النص في القالب
```django
{% load i18n %}
<h1>{% trans "النص الجديد بالعربية" %}</h1>
```

### الخطوة 2: إضافة الترجمة
افتح ملف: `locale/en/LC_MESSAGES/django.po`

أضف:
```po
msgid "النص الجديد بالعربية"
msgstr "New Text in English"
```

### الخطوة 3: تجميع الترجمات
```bash
python compile_translations.py
```

### الخطوة 4: أعد تشغيل السيرفر
```bash
python manage.py runserver
```
