# نظام التصميم - مؤسسة الكتيع للتجارة والاستيراد (الهوية الجديدة)

## 🎨 نظام الألوان (Color Palette)

### الألوان الأساسية (Primary Colors)

#### 1. كحلي داكن فاخر (Navy Premium)
```css
--primary-navy: #16263F
```
- الاستخدام: الهيدر (في بعض الأقسام)، الفوتر، الأقسام الرئيسية المميزة (Hero Section)، خلفيات الـ CTA.
- الخصائص: يعكس القوة، الثبات، والموثوقية الدولية.

#### 2. ذهبي ملكي (Royal Gold)
```css
--primary-gold: #C6A75E
```
- الاستخدام: العناوين البارزة، الأيقونات، حدود العناصر المميزة، الأزرار.
- الخصائص: يعكس الفخامة العملية والتميز.

#### 3. ذهبي فاتح (Gold Light)
```css
--gold-light: #E2C889
```
- الاستخدام: التدرجات، حالات الـ Hover، العناصر الثانوية.

### الألوان الثانوية (Secondary Colors)

#### 4. رمادي دافئ (Warm Gray)
```css
--bg-gray: #F4F4F4
```
- الاستخدام: خلفيات الأقسام الثانوية، البطاقات.

#### 5. جرافيت (Graphite)
```css
--text-graphite: #2B2B2B
```
- الاستخدام: النصوص الأساسية والمحتوى الطويل.

## 🎨 التدرجات (Gradients)

### التدرج الذهبي الفاخر
```css
background: linear-gradient(135deg, #C6A75E, #E2C889);
```
- الاستخدام: الأزرار الذهبية، الأيقونات المميزة.

### تدرج الكحلي العميق
```css
background: linear-gradient(135deg, #16263F, #1d3354);
```
- الاستخدام: الأقسام الخلفية، الهيدر.

## 📝 نظام الخطوط (Typography)

### للغة العربية
- الخط الأساسي: **Cairo**
- العناوين: **Cairo Bold** (700-800)
- المحتوى: **Cairo Regular** (400-500)

### للغة الإنجليزية
- الخط الأساسي: **Montserrat** أو **Inter**
- الطابع: عصري، مباشر، واحترافي.

---

## 🎯 أسلوب التصميم (Design Direction)

### الشخصية البصرية
- ✅ **فخامة عملية**: توازن بين المظهر الفاخر والطابع التجاري الجاد.
- ✅ **موثوقية**: استخدام الكحلي الداكن كركيزة أساسية للتصميم.
- ✅ **وضوح**: مساحات بيضاء منظمة لتسهيل القراءة والوصول للمعلومات.
- ✅ **زوايا احترافية**: (border-radius: 6px) تعطي طابعاً هندسياً منظماً.

---

## 🔲 العناصر الأساسية (Components)

### الأزرار (Buttons)

#### الزر الذهبي (الأساسي)
```css
.btn-gold {
    background: linear-gradient(135deg, var(--primary-gold), var(--gold-light));
    color: var(--primary-navy);
    font-weight: 700;
    border-radius: 6px;
}
```

#### الزر الكحلي (الثانوي)
```css
.btn-primary {
    background: var(--primary-navy);
    color: var(--primary-gold);
    border: 1px solid var(--primary-gold);
}
```
